Path Finding Visualizere Project

 May 2022

This path finding visualizer project uses a* algorithm. This project is a path finding visualizer tool to visualize the a* algorithm as it runs. This algorithm is an informed search algorithm which is much more efficient than standard algorithms like breadth first search and depth first search. The project works as, User can click on each cell to make it a wall, a source, or a destination. Once that happens, the pathfinding algorithm starts, adding nodes to closedList and openList accordingly and changing colors of cells in the two lists. Finally, when the path is found the grids which are part of this path are coloured in a different tone.
